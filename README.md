# מדריך הפעלה — בוט הטופס בטלגרם 🤖⚽

הבוט מושך משחקים ויחסים אמיתיים של היום מ-API-Football, ובונה טופס
לפי רמת סיכון, סכום ויחס שאתה בוחר. הכל עובד 24/7 בענן חינמי.
לא צריך לדעת לתכנת — רק להעתיק ולהדביק.

---

## שלב 1: ליצור את הבוט בטלגרם (טוקן)

1. בטלגרם, חפש את **@BotFather**.
2. שלח `/newbot`, תן שם, ותן שם משתמש שמסתיים ב-`bot`.
3. תקבל **טוקן** כמו `7123456789:AAH...xyz` — שמור אותו בצד.

---

## שלב 2: להשיג מפתח API למשחקים (חינם)

1. היכנס ל-https://dashboard.api-football.com/register והירשם (חינם, בלי כרטיס אשראי).
2. אחרי אימות המייל, בלוח הבקרה תמצא **API Key** — מחרוזת ארוכה.
3. שמור אותה בצד. (התוכנית החינמית נותנת ~100 בקשות ביום — יותר ממספיק.)

> הערה: היחסים (odds) בתוכנית החינמית זמינים למשחקים מסוימים בלבד.
> אם למשחק אין יחס, הבוט עדיין ישים אותו בטופס עם ערך ברירת מחדל.

---

## שלב 3: להעלות את הקוד ל-GitHub

1. היכנס ל-https://github.com והירשם אם צריך.
2. **New repository** → תן שם (למשל `my-tofes-bot`) → **Public** → **Create**.
3. לחץ **"uploading an existing file"** וגרור את שלושת הקבצים:
   - `bot.py`
   - `requirements.txt`
   - `README.md`
4. לחץ **Commit changes**.

---

## שלב 4: להריץ בענן חינם עם Render

1. היכנס ל-https://render.com והירשם עם GitHub.
2. **New +** → **Background Worker** (חשוב: Background Worker, לא Web Service).
3. חבר את ה-repository שיצרת.
4. מלא:
   - **Build Command**: `pip install -r requirements.txt`
   - **Start Command**: `python bot.py`
   - **Instance Type**: Free
5. גלול ל-**Environment Variables** והוסף **שניים**:
   - Key: `BOT_TOKEN`         | Value: הטוקן מ-BotFather
   - Key: `FOOTBALL_API_KEY`  | Value: המפתח מ-API-Football
6. לחץ **Create Background Worker**.

אחרי דקה-שתיים תראה בלוגים `Bot is running...`

---

## שלב 5: להשתמש

1. פתח את הבוט בטלגרם, שלח `/start` ואז `/new`.
2. בחר רמת סיכון → סכום → יחס.
3. הבוט שולף את משחקי היום ובונה טופס.

- סיכון גבוה = יותר משחקים בטופס + יותר הפתעות (2/X).
- סיכון נמוך = פחות משחקים + פבוריטים (היחס הנמוך בכל משחק).
- `/cancel` — לבטל באמצע.

---

## אם משהו לא עובד
- הבוט שולח "רשימת גיבוי"? כנראה שהמפתח `FOOTBALL_API_KEY` שגוי או שאין משחקים היום — בדוק אותו ב-Render.
- הבוט לא מגיב? בדוק בלוגים של Render שכתוב `Bot is running...` ושה-`BOT_TOKEN` נכון.
